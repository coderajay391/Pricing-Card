## Features

- Clean pricing layout (plan name, price, description)
- Highlight / “Most Popular” style support (via props/classes)
- Feature list with check icons
- Responsive design

## Folder

- `frontend/pricingCard/` — contains the UI implementation and related assets.

## Customization

- Update colors / typography via CSS or Tailwind utility classes (depending on the implementation).
- Add or remove features by editing the `features` array.

## Notes

- This module is intended as a UI building block.
- No backend/database is required.
